# DiaMove Simulator

運転整理ダイヤ（列車の運行計画）を、鉄道の**配線図の上で時系列アニメーション**として再現する、自己完結型の可視化ツールです。

`route_map.pdf`（配線図）と `tmp.xls`（ダイヤデータ）から、単一の HTML ファイル `diamove_sim.html` を生成します。生成された HTML は外部リソースに依存しないため、そのままブラウザで開くだけで動作します。

## 特徴

- **配線図はそのまま** — `route_map.pdf` をラスタライズして背景に埋め込むため、配線図の見た目は元 PDF と完全に一致します。
- **列車位置は線路上に正確配置** — PDF から抽出した各ブロックのラベル座標に基づき、列車を線路上に重ねて表示します。
- **編成数を自動検出** — `tmp.xls` の列数から編成数を判定するため、編成を増減したダイヤでも再実行するだけで対応できます。
- **単一ファイルで完結** — 背景画像も Base64 で埋め込まれた自己完結 HTML なので、配布・共有が容易です。
- **操作 UI 付き** — 再生／一時停止（スペースキー可）、先頭へ戻る、時刻シーク、再生速度（1〜600 倍）、編成の色凡例。

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `build_sim.py` | ビルドスクリプト。入力を読み込み `diamove_sim.html` を生成する。 |
| `route_map.pdf` | 配線図（ベクター）。背景画像とブロック座標の抽出元。 |
| `route_map_graph.py` | ブロックの隣接リスト `l` と駅ブロック `st` の定義。 |
| `tmp.xls` | ダイヤデータ（拡張子に反して**タブ区切りテキスト**）。 |
| `diamove_sim.html` | 出力される自己完結シミュレーション（生成物）。 |
| `requirements.txt` | Python 依存パッケージ。 |

## 必要環境

- Python 3
- [PyMuPDF (fitz)](https://pymupdf.readthedocs.io/) — PDF のラスタライズと座標抽出に使用
- Pillow

```bash
pip install -r requirements.txt
```

## 使い方

```bash
python build_sim.py
```

実行すると、検出した編成数・背景画像サイズ・ステップ数などをログ出力し、`diamove_sim.html` を生成します。生成された HTML をブラウザで開けばシミュレーションが再生できます。

ダイヤ（`tmp.xls`）や配線図（`route_map.pdf`）を更新した場合は、`build_sim.py` を再実行するだけで反映されます。

## データ形式

### `tmp.xls`（タブ区切りテキスト）

1 行 = 1 ステップ（15 秒 = 0.25 分刻み）。列構成は編成数を `N` として **`2N + 3` 列**：

| 列 | 内容 |
| --- | --- |
| 0 | 時刻 [分] |
| 1 〜 N | 各編成の「左端駅からの距離」 |
| N+1 | 空（区切り） |
| N+2 〜 2N+1 | 各編成のブロック番号 |
| 2N+2 | 空（末尾） |

ブロック番号は `route_map_graph.py` のノード番号（1〜140）に対応します。

### `route_map_graph.py`

- `l` — 1 始まりノードの隣接リスト（配列自体は 0 始まりインデックス）。
- `st` — 駅ブロックの組 `[1, 2, 17, 18, ...]`（駅①〜⑨）。

### 配線の位相（概要）

- **本線（複線）**: 上り＝奇数ブロック（1〜121）、下り＝偶数ブロック（2〜122）。
- **渡り線**: 3-5 / 4-6、117-119 / 118-120。
- **支線**: 55/56 → 127 → 128（行止り）。
- **車庫**: 79/80/83/84 → 123/124 → 125 → 126-129-…-134 → 扇形（135〜140）。

## カスタマイズ

`build_sim.py` 内の主なパラメータ：

- `R` — 背景ラスタライズ倍率（解像度）。
- `CLIP` — PDF から切り出す領域。
- `TOPY` / `BOTY` — 本線（上り／下り）の線路 y 座標。
- HTML 内の描画部（列車の円）: 半径 `r`（`render()` 関数）で列車マーカーの大きさを調整。
- `#canvasWrap { height: ... }` でキャンバスの表示高さを調整。
